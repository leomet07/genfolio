# Genfolio Server

This is the Genfolio server, which is based on the [Express.js](https://expressjs.com/) API framework.


## Environment Variables

To run this project, you may wish to add the following environment variable to your .env file

`GH_TOKEN` - A GitHub API access token. (optional)

## Get started

Install the dependencies...

```bash
npm install
```

...then start the Genfolio API server:

```bash
npm start
```

Navigate to [localhost:5678](http://localhost:5678).

## Deploy Script
Alternatively, you could use the `redeploy.sh` script at the root of this repository, which handles installation and setup with an optimized production build.
